Welcome to {{ title }} play ground!

First button toggle a div using ngxCacheIf directive.

Second button toggle a div using legacy ngIf directive.

Each button triggers a view with content displayed for 5 seconds from the ngOnInit lifecycle event.
The first button will toggle a div with content cached. The test case aim to show how the content is rendered only once using ngxCacheIf. If you click multiple time you won't see the ngOnInit message.
If you click multiple times on the ngIf legacy button the element is rendered each time we click on the button. There is no cache of the previously rendered component. So each time you will see the ngOnInit message.
